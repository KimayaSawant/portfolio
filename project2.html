Food Classifier and Calorie Counter

What is it?

A machine learning web application that uses image recognition and computer vision to classify food types and calculate their nutritional content. This tool helps users track their calorie intake efficiently.

What was this built with?

This project leverages Python for backend logic, TensorFlow for machine learning, HTML and CSS for frontend design, and MySQL for database management.

What were the biggest challenges?

- Building an accurate image recognition model for classifying food items.
- Reducing execution time while maintaining high accuracy.
- Designing a user-friendly interface for non-technical users to interact with the application.

What did I learn?

I gained experience in applying computer vision and machine learning techniques to practical problems. Additionally, I learned the importance of optimizing execution time and collaborating effectively in a team environment.
