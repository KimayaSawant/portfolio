Enhanced Vision-Based Detection System

What is it?

A real-time detection system to enhance pedestrian safety at urban crosswalks. The system identifies pedestrians using advanced machine vision and anonymizes individuals to protect privacy.

What was this built with?

The project leverages NVIDIA Jetson Orin Nano for real-time processing, OpenCV for computer vision tasks, CUDA for GPU acceleration, and the YOLO deep learning model for detection.

What were the biggest challenges?

- Balancing real-time processing with high accuracy in detection.
- Ensuring privacy protection while handling video data.
- Deploying the system on constrained hardware like Jetson Orin Nano.

What did I learn?

This project expanded my knowledge of computer vision, deep learning, and edge computing. It taught me how to design and optimize solutions for real-time applications in constrained environments.
