Petstagram Mobile App

What is it?

Petstagram is a dynamic social media app designed for pet enthusiasts. The app allows users to share photos, explore pet-related content, and interact with other pet lovers in real-time. Itâ€™s a space for pet owners to connect and showcase their furry friends.

What was this built with?

The project was built using Flutter for a seamless cross-platform mobile experience. Firebase was used for real-time database management, and Google Cloud Platform (GCP) provided scalable cloud storage for user-generated content.

What were the biggest challenges?

- Ensuring real-time updates for a smooth user experience.
- Managing and scaling cloud storage to handle large volumes of media uploads.
- Designing an intuitive UI/UX that appeals to a diverse audience of pet enthusiasts.

What did I learn?

This project enhanced my understanding of mobile app development with Flutter. I also learned about integrating Firebase and GCP for scalable backend solutions. Additionally, I gained experience in designing engaging user interfaces for a social media application.
